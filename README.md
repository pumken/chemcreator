# ChemCreator

A text-based tool for identifying organic molecules.

Written in Rust with the [`ruscii`](https://github.com/lemunozm/ruscii) library.

## Features

- [x] Easy-to-use text-based UI
- [ ] Variable-size grid

### Supported Molecule Types

- [x] Alkanes
- [x] Alkenes
- [x] Alkynes
- [ ] Alcohols
- [ ] Aldehydes
- [ ] Carboxylic acids
- [ ] Ethers
- [ ] Ketones
- [ ] Halogenoalkanes
- [ ] Esters
